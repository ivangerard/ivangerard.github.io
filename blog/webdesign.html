AWD (Adaptive Web Design) VS RWB (Responsive Web Design)

by Ivan Gerard

Posted on August 6, 2016 at 9:51 PM

  1. 1. Apa itu adaptive dan responsive web design? Apakah kamu setuju bahwa teknik tersebut bagus? Mengapa atau mengapa tidak?

    Adaptive merupakan static layout , kalau responsive more like dynamic. Jika ingin peformance yang cepat, adaptive lebih baik. tapi effort untuk membuatnya lebih besar. Responsive peformancenya lebih lambat jika dibandingkan Adaptive, namun pengerjaannya lebih cepat dibandingkan adaptive. Google ajah recommended Responsive instead of adaptive

    2. Apakah hadirnya smartphone dan tablet mengubah desain web secara signifikan? Bagaimana kira-kira teknologi baru akan mengubah desain web di masa depan?

    Jika dilihat trend nya kehadiran smartphone tidak akan mengubah design web secara significant, dengan memnggunakan responsive web design kita tidak perlu melakukan coding untuk bebrbagai ukuran layar. Sehingga memudahkan coder untuk membuat design untuk pengguna smartphone maupun tablet.

    3. Bagaimana kira-kira orang yang memiliki kekurangan fisik menggunakan website? Apakah kita perlu menyesuaikan website untuk mereka juga? Solusi apa yang dapat mengatasi keadaan tersebut?

    Untuk teman-teman yang memiliki kekurangan fisik, mungkin pada websitenya harus diberikan bantuan untuk fitur suara, atau fitur lainnya.

    Ide atau konsep apa yang sesuai denganmu saat mempelajari desain web?

    Responsive design pastinya.
